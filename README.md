# Высокоскоростная реализация шифра "Кузнечик"

### В данном репозитории был реализован ГОСТ Р 34.12-2015 (шифр Кузнечик) Признаный государственным стандартом РФ разработанный ИнфоТеКС и ФСБ РФ.
#### Шифр является блочным, размер блока составляет 128-бит, размер мастер-ключа 256-бит, размер раундового ключа 128-бит.

### Benchmark
- **Процессор: 11th Gen Intel(R) Core(TM) i5-1135G7 @ 2.40GHz (2.42 GHz)**
- *Скорость 1.2ГБ/С ( Размер файла 1.31ГБ )*
- *Скорость 1.5ГБ/С ( Размер файла 6.8ГБ )*
## Что реализовано

### AVX2 SIMD

- использование 256-битных регистров
- обработка 8 блоков за итерацию
- ускорение линейных операций

### Precomputed Tables

- **LS_TABLE** — объединение S и L преобразований
- **L_COEFFS** — ускорение R-преобразования

### Multithreading

- OpenMP
- `#pragma omp parallel for`
- использование всех ядер CPU
  
### Кроссплатформенность
- Windows
- Linux

# Как запускать?
**Linux**
- ```g++ main.cpp kyznechik.cpp pkcs.cpp -o main.exe -O3 -mavx2 -fopenmp```

**Windows**
- ```g++ main.cpp kyznechik.cpp pkcs.cpp -o main.exe -O3 -mavx2 -fopenmp -municode```

**Drag-On-Drop**
- **Просто перетяните ваш файл или директорию на main.exe и он будет успешно зашифрован.**
